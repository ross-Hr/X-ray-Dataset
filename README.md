# X-ray-Dataset
This repository contains two datasets, OPIXray and CLCXray.

OPIXray can be downloaded from https://drive.google.com/file/d/1py2UJSzEEGR66jJ447lTgbOjF75sU6yp/view?usp=sharing 

CLCXray can be downloaded from https://github.com/GreysonPhoenix/CLCXray.git

DVXray can be downloaded from https://github.com/Mbwslib/DvXray
